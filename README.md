👑 합격의 신 : 나는 어디로? (2026 대입 시뮬레이션)

"내 성적으로 갈 수 있는 대학은 어디일까?" > 2026학년도 대학 입시를 미리 체험해보는 실전형 대입 시뮬레이션 게임입니다.

📖 프로젝트 소개 (Introduction)

**[합격의 신]**은 실제 입시 트렌드와 데이터를 기반으로 구성된 웹 시뮬레이션 게임입니다.

플레이어는 수험생이 되어 내신과 모의고사 성적을 입력하고, 퀴즈와 대학별 고사(면접, 논술, 실기)를 거쳐 수시 6장과 정시 3장의 원서를 접수하며 최종 합격을 목표로 합니다.

단순한 운이 아닌, **전형별 특징(교과/종합/논술)**과 지원 전략(상향/적정/안정), 그리고 전공 적합성이 중요한 전략 게임입니다.

✨ 주요 기능 (Key Features)

1. 🏫 방대한 대학 데이터베이스 (60+ Universities)

SKY (최상위권): 서울대, 연세대, 고려대

인서울 주요대: 서강대, 성균관대, 한양대, 중앙대, 경희대, 이화여대 등

지거국 (지방거점국립대): 부산대, 경북대, 전남대, 충남대 등 10개교

수도권 및 지역 주요 대학: 가천대, 경기대, 한경국립대, 대진대, 협성대, 백석대 등

전형별 커트라인: 인문/자연/예체능 계열별 차등 적용 및 수능 최저학력기준 구현

2. ⚡ 스탯 향상 시스템 & AI 추천

성적 향상 퀴즈: 국어, 수학, 영어, 사탐, 과탐, 시사 등 40+ 문제 풀 (콤보 보너스 시스템)

AI 원서 추천: 플레이어의 성적과 대학 입결을 분석하여 합격 확률이 높은 대학에 [👍 AI 추천] 배지 표시

실시간 피드백: 퀴즈 정답/오답 시 내신과 수능 점수의 등락을 즉시 시각화

3. 🎤 고도화된 대학별 고사 (Mini Games)

심층 면접 (학생부종합): 희망 학과를 분석하여 기본 → 상황 → 전공 심화 3단계 질문 제시

고난도 논술 (논술전형): 딜레마, 수리, 철학 등 사고력을 요하는 문제 출제

실기 고사 (예체능): 반응 속도 테스트를 통한 실기 점수 산출

4. 🔄 N수생 시스템 (Realism)

재수 모드: 불합격 시 재수 선택 가능 (내신 -0.5등급 페널티 적용 후 리셋)

군수 모드: 군대에서 수능 준비 (내신 -0.7등급 페널티, 하지만 퀴즈 기회 2배 제공)

데이터 저장: 브라우저 종료 후에도 N수생 신분 유지

5. 🎨 큐티뽀짝 UI & 사운드

디자인: 파스텔 톤의 귀엽고 직관적인 UI/UX

사운드: 합격/불합격 시 기분 좋은(혹은 슬픈) 효과음 재생 (Web Audio API)

🕹️ 게임 방법 (How to Play)

프로필 생성: 이름, 출신 고교, 희망 학과, 성적(내신/모의고사)을 입력합니다.

스탯 업(Quiz): 5과목(군수생은 10과목)의 퀴즈를 풀어 성적을 끌어올리세요. 연속 정답 시 보너스!

수시 지원: 60여 개 대학 중 최대 6곳을 골라 지원합니다.

Tip: 교과 전형은 최저학력기준을, 종합/논술은 면접/논술 실력을 믿고 상향 지원해보세요.

대학별 고사: 지원한 전형에 따라 면접장, 논술고사장, 실기장으로 이동하여 미니게임을 진행합니다.

수시 발표: 합격 시 최종 등록(게임 종료), 불합격 시 정시로 이동합니다.

정시 지원: 가/나/다군 총 3장의 카드로 마지막 승부를 봅니다.

🛠️ 기술 스택 (Tech Stack)

Core: HTML5, CSS3, JavaScript (Vanilla JS)

Styling: Tailwind CSS (CDN)

Icons: Font Awesome

Font: Google Fonts (Jua, Gaegu, Noto Sans KR)

Audio: Web Audio API (No external files required)

🚀 설치 및 실행 (How to Run)

이 프로젝트는 별도의 서버 설치 없이 브라우저에서 바로 실행 가능합니다.

이 저장소(Repository)를 Clone 하거나 ZIP으로 다운로드합니다.

index.html 파일을 크롬(Chrome), 사파리(Safari), 엣지(Edge) 등 최신 브라우저에서 엽니다.

GitHub Pages 설정을 통해 웹 호스팅이 가능합니다.

Settings -> Pages -> Branch: main -> Save
